About

For my website, I create a model website which includes a gun model, basketball model and a cup model

3D models

For this website, I have created a 3D models of a 9mm Beretta, Basketball and Coke can in blender. In Blender I created an shoot animation for gun where the gun can shoot. For basketball, I created an animation where ball goes into the net as you click dunk buitton Also, for Coke can I created an open animation for the can

Design

I created the Gun, Basketball and Can in a bootstrap canvas. I used bootstrap to create a Navbar. I used CSS to create an interactive design. I used three.js to light my scene with a Hemisphere Light and add a light GUI spot light to allow users to control the lighting of the can, bottle and glass. I add the models into my web page using three.js. I added buttons to control the animation like for , gun shooting; for Basketball, Dunking; for Cup, Jumping. Also there are two commons buttons for all models to control the wireframe and rotation.

Integration

I have downloaded the audio online and sync it with the model so that it played in sync with the animations when they loaded and played in my web page. I have added images into my landing page and using CSS the image resizes as the screen size resizes.

Interaction

I added orbit controls to move the can in the web page.
